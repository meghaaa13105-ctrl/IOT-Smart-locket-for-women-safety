# IoT Smart Locket for Women Safety

An IoT-based smart safety locket designed for women's security.  
The device uses sensors and microcontroller technology to detect emergency situations and send alerts.

## Features
- Emergency alert system
- GPS/location tracking
- IoT-based real-time monitoring
- Compact wearable device

## Tech Used
- Arduino / ESP32
- Embedded C
- IoT Sensors
- GPS module

## Purpose
To provide a quick-response safety device for women during emergency situations.
